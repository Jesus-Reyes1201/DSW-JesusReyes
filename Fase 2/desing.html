Flex Box y CSS Grid

Esta pajina es con el fin de conocer, analisar y aprender sobre los modos de diseño mencionados anteriormente,
buscando aplicarlos en la creacion de sitios web el cual es el objetivo de aprendizaje de esta materia.

Flex box

Es un modulo de diseño que se creo para mejorar la forma en que se hace diseño responsive,
utilizando menos codigo y facilitando el posicionamiento de elementos.

CSS Grid

Es un modelo de maquetacion en base a una rejilla que ofrece control total,
ubicacion de los items es libre o automatico, comportamientos con atributos novedosos.
